### Installation du projet

```bash
  git clone https://iut-info.univ-reims.fr/gitlab/mena0018/api-bookmark.git
  cd api-bookmark/
  composer install
```

### Configuration de la base de données

```bash
  composer db
```

### Lancement de l'application

```bash
  composer start
  http://127.0.0.1:8000/   
```
